# Docs

Central de documentação do projeto. Estrutura esperada:

- `docs/adr/` — ADRs.
- `docs/AGENT_CONTEXT/` — contexto do agente e tasks operacionais.
- `docs/sprints/` — notas de sprint e checklists.
- `docs/epics/` — epics e histórias (cada épico tem seu próprio `.md` em `docs/epics/histories/`).

Atualize `docs/DOCS_INDEX.md` ao adicionar ou mover documentação.
